# B2 · 品类独创——市场如何认得我

## 1. 旧认知隔离

很多人以为品类独创就是起一个好听的名字。但它不是命名游戏——名字只是最后一厘米，前面九十九米是把甜用户、贵任务、心理账户一层层挖出来；没有这些地基，再惊艳的名字也只是空转的修辞。

很多人以为品类独创就是包装卖点、做差异化话术。但话术层面的差异化，竞品一个月就能抄走；品类层面的重新定义，抄走的人反而在替你教育市场。

很多人以为品类独创就是「比竞品更好」。但这恰恰是最危险的思路——「更好」意味着你接受了旧品类的赛道和评分标准，从此在别人的地图上打仗，卷入同质化与价格战。品类独创的野心不是当第一名，而是当唯一一个。

## 2. 小切口

一位把寺院文化做成随身产品的创始人，最初最头疼的不是销量，是回答「你这是什么」。渠道问他：算文创？算礼品？算用品？哪个分类都装不下，哪个货架都摆不对——产品无法分类，意味着系统里没有它的位置。

转折点是他不再试图挤进旧分类，而是反过来把「无法分类」当成信号：既然旧品类解释不了它，那就重新定义一个新品类，让用户第一次听到就明白「这是给我的」，并且知道该从哪个预算里掏钱。一年半后，这个说不清算什么的产品，做到了年流水数千万。

同样的剧本在别处重演：一位做青少年 AI 教育的创始人没有挤进「培训班」，而是开创新赛道，用户自带教育预算入场；一位企业健康服务创始人不卖体检卖安心，高毛利不来自成本控制，来自心理账户的切换。三个行业，同一个动作：不进旧货架，自己开货架。

## 3. 大纵深

**脑科学维：心智货架只有七个位置。** 现场讲授引脑科学的结论：一个品类里，大脑大约只装得下七个品牌（Get笔记 note 1911384455311537296）。挤进一个已满的货架，你要对抗的是人类记忆的硬件限制。而大脑对三样东西天然敞开：新奇性（注意力捕获）、因果叙事（意义构建）、身份认同（传播驱动）——品类独创恰好同时按下这三个开关：新品类天然新奇，重新定义自带叙事，「我用的是独一份」自带身份。

**商业维：跟随者的代价是指数级的。** 波士顿咨询的测算：同一品类里，老二要用老大四倍的资源才能追平，老四要八倍（Get笔记，同上）。而 721 法则宣判了结局：独创品类者拿走七成财富（Get笔记 note 1903664971003368064）。两组数字拼在一起就是品类独创的全部商业逻辑——当跟随的成本是指数级、独创的回报是七成时，「再开一个货架」不是浪漫，是算术。

**理论维：定义权即主权。** 当用户说不出「你像谁」，比价就失去了对象，预算从「比价账户」搬进「唯一账户」。这背后是心理账户理论：人不是用一个总钱包做决策，而是把钱预先分进功能、情绪、社交、投资四个账户，每个账户的价格弹性完全不同。品类独创的深层动作，是把你的产品从卷价格的功能账户，搬进为意义付费的账户里。

## 4. 新植入

正典一句话定义：

> 品类独创，是把原力翻译成市场能识别、能记住、愿意付费、愿意转述的新入口。

第一性问题：**市场如何认得我？**

十六字心法：**无法分类、重新定义、自带预算、品类独创**（现场讲授版第四拍亦作「品类共创」——独创立入口，共创养生态；Get笔记 note 1911384455311537296）。

落地公式：

> **三角定位 = 甜用户 × 贵任务 × 入脑表达**

四大心理账户：

| 账户 | 用户为什么付费 | 特征 |
|---|---|---|
| 功能账户 | 解决实际问题 | 有上限，容易卷价格 |
| 情绪账户 | 被看见、被安放、被疗愈 | 溢价空间大 |
| 社交账户 | 身份、圈层、面子、归属 | 自带传播 |
| 投资账户 | 未来收益、能力增值、机会放大 | 预算弹性最大 |

原子模块（完整，不删减）：

| 模块 | 名称 | 内容 |
|---|---|---|
| B2.1 | 甜用户 | 谁最自然需要这个原力 |
| B2.2 | 贵任务 | 什么问题值得用户付费解决 |
| B2.3 | 心理账户 | 用户把这件事放在哪个预算和意义账户 |
| B2.4 | 旧品类对标 | 用户原来如何理解这类问题 |
| B2.5 | 新品类命名 | 我不是旧方案，而是什么新方案 |
| B2.6 | 入脑表达 | 一句话让人理解、记住、转述 |

判断标准（完整，不删减）：

| # | 判断标准 |
|---|---|
| 1 | 用户是否一听就懂？ |
| 2 | 是否能转述？ |
| 3 | 是否知道为什么付费？ |
| 4 | 是否能把我和旧方案区分开？ |

通关五检：① 十六字心法四维全过；② 甜用户四维筛选通过、S 级客户 ≥ 5；③ 贵任务进入情绪/社交/投资账户；④ 30 秒电梯演讲通过、一句话品类定位清晰；⑤ MVP 闭环跑通、出现真实转化。

**输出**：新品类定义、一句话价值主张、心理账户定位、品类命名候选、内容表达框架。**下一步接口**：进入 [B3 · 模式升维](23-B3-模式升维.md)。**回写位置**：OS 文脉词典、用户语言库、内容主题库、品牌叙事库。

品类独创解决的不是「怎么把产品说得更好」，而是「让市场用一个全新的入口认出我」。

今天就做：填一张三角定位卡——谁最需要我且最愿意付费（甜用户）？他最贵的问题落在哪个账户（贵任务）？我如何让他 30 秒听懂并愿意复述（入脑表达）？

## 5. 高弧光

品类独创最容易被误解成营销技巧，但它本质上是一次翻译的诚实劳动：把你在 [B1](21-B1-原力借势.md) 确认的势、在《原力资产》里淬炼出的原力，翻译成市场的母语。翻译得好不好，检验标准不在你嘴上，在用户嘴上——他能不能不看你的资料，用自己的话把你讲给朋友听。

「被认出」是一个比「被知道」深得多的状态。被知道靠曝光，被认出靠位置：你在他的心智里有了一个专属座位，从此他遇到那个问题，第一个想起的名字是你。这一关交付的获得物叫心理账户——一个开在用户头脑里的户头，它是下一关利润容器的进水口。

## 6. 可复用金句

1. 进不了前七的心智货架，就自己开一个货架。
2. 当用户说不出你像谁，预算就从比价账户搬进唯一账户。
3. 老二要花四倍资源，老四要花八倍——独创者不用追赶，因为定义权在他手里。
4. 「更好」是在别人的地图上打仗，「唯一」是自己画地图。
5. 品类独创不是给产品起名，是替用户重新理解他自己的问题。
