# 原力叙事四拍：小切口，大纵深，新植入，高弧光

> 本文是这座知识库「表达系统的自述」。库内所有概念叙事篇——从[母公理ʸx](../00-canon/01-母公理ʸx.md)到十二模块——都按本文方法写成。读懂这一篇，就读懂了其余每一篇的骨架。姊妹篇：[叙事质量八标准与禁区](42-叙事质量八标准与禁区.md)。

## 1. 核心公式与一句话定义

核心公式：

> **原力叙事 = 心智隔离 × 小切口 × 大纵深 × 新植入 × 高弧光 × 可回写**

对外正典句（八字诀）：**小切口，大纵深，新植入，高弧光。**

一句话定义：

> 原力叙事，是用小切口打开共情，用大纵深建立信任，用新植入重塑心智，用高弧光唤醒一个人重新出发的原力。

最短表达：**小切口打开人，大纵深说服人，新植入改变人，高弧光照亮人。**

注意公式里的乘号：六个因子是相乘关系，不是相加。任何一项为零，整篇叙事为零——有切口没纵深是故事会，有纵深没植入是材料汇编，有植入没弧光是干货贩卖，全都有但不可回写，则是一次性的表达，违背[母公理ʸx](../00-canon/01-母公理ʸx.md)。

## 2. 最高原则：叙事是心智迁移，不是表达观点

原力叙事的最高原则只有一条：**叙事不是表达观点，而是重塑心智**——让听众在一篇内容里完成一次心智迁移。

普通表达问一个问题：「我想说什么？」原力叙事问七个问题（心智迁移七问）：

1. 听众**原来**是怎么理解这件事的？
2. 这个旧理解如何**限制**了他？
3. 如何先**隔离**他的旧认知，腾出心智空间？
4. 用哪个**小切口**打开他的心门？
5. 用什么**大纵深**让他相信这是结构性规律，不是个人观点？
6. 植入什么**新定义**，让他从此换一副眼镜看这件事？
7. 最后带他回到哪种**成长初心与原力觉醒**？

七问的顺序就是叙事的顺序：先拆旧屋，再开门，再打地基，再立新柱，最后点灯。

## 3. 底层机制：心智隔离

四拍开打之前有一道地基工程——**心智隔离**。它源于一个认知事实：听众面对新概念的第一反应，是把它塞进旧抽屉。「哦，ʸx 不就是复利嘛」「原力不就是热情嘛」——一旦塞进旧抽屉，后面说什么都白说，因为他以为自己早就懂了。

心智隔离要在开口之初讲清五件事：

- 它**不是什么**；
- 它容易被**误解成什么**；
- 它和旧概念的**边界**在哪里；
- 为什么旧认知**不够用**；
- 继续用旧认知会**错过什么**。

标准句式：

> 「很多人以为 X 是___。但 X 不是___。不是___。也不是___。」

关键分寸：隔离的目的**不是攻击旧认知，而是腾空旧心智**。旧认知在它的时代是对的，只是装不下新东西了。语气是「那个抽屉太小」，不是「你以前全错了」。

## 4. 四拍逐拍详解

### 第一拍 · 小切口——打开人

**定义**：一个极小、极热、极具体的入口。可选素材：亲身经历、学员瞬间、一段真实对话、一个创业者的困境、一个热点、一处高共情细节、一组时代反差、一幅生活画面。

**任务**：打开心门——不是讲完故事。切口只负责让人愿意听下去，故事的完成留给后面的拍子。

**判断标准**：听众在第一分钟里心里冒出一句「**这说的是我**」。做不到这一点，切口再精彩也只是别人的故事。

**标准句式**：

> 「我第一次真正意识到这个问题，是因为___。」
> 「有一次，一个创业者跟我说：___。」

### 第二拍 · 大纵深——说服人

**定义**：从小故事打穿到结构。五个纵深维度，每篇取二到四维：

| 维度 | 内容 |
|---|---|
| 时代纵深 | AI、技术革命、康波长波、产业迁移、范式转换 |
| 历史纵深 | 商业史、文明史、组织演化史 |
| 理论纵深 | 战略学、心理学、组织学、认知科学、复杂系统、叙事经济学 |
| 商业纵深 | 硅谷与产业案例、创业实践、用户心智 |
| 个人纵深 | 亲历、学员案例、人生转折 |

**任务**：让听众意识到——「这不是我一个人的问题，是时代、结构、心智、商业系统共同作用的结果。」个体困境一旦被放进结构里，羞耻感变成洞察力，防御变成好奇。

**判断标准**：听众从「他遇到过」升级为「原来所有人都会撞上这堵墙」。

**标准句式**：

> 「如果把这个问题放大看……它背后有一个更大的时代结构：___。过去___。今天___。未来___。」

### 第三拍 · 新植入——改变人

**定义**：在旧认知已隔离、心门已打开、纵深已建立之后，植入新的认知资产——新定义、新公式、新模型、新判断、新分类、新路径或新行动框架。

**任务**：回答四件事——过去误解了什么；现在如何重新定义；新定义带来什么新判断；新判断带来什么新行动。四问答不全，植入就不完整。

**判断标准**：新植入必须同时做到——清晰、锋利、**可复述**、可迁移、可行动、与旧认知形成明确区隔，并且服务于三部曲正典龙骨（见第 7 节）。「可复述」是硬门槛：听众第二天能用一句话转述给别人，才算植入成功。

**标准句式**：

> 「所以 X 真正是：X =___。它解决的不是___，而是___。它把我们从___，带向___。」

### 第四拍 · 高弧光——照亮人

**定义**：叙事的终点是**价值复位**，不是煽情。内容不能只落到方法、增长、赚钱、效率、工具，必须回到成长、初心、少年英雄、正心正念、真善美、原力觉醒。

**任务**：让听众合上内容时的感受不是「我学到一个方法」，而是——「原来我可以重新理解自己」「我的困境不是孤立的」「我身上最初的那点光，还可以进入世界，并且持续生长」。

**判断标准**：弧光落点在人不在术。方法讲完之后，还有没有一段话是写给那个「最初出发的人」的。

**标准句式**：

> 「原力不是让你变成另一个更厉害的人。原力是让你重新回到自己，并把那个最初的光，训练成可以进入市场、服务他人、也持续保护自己的事业系统。」

## 5. 标准工作流：八步

1. **识别主题**——这篇要迁移的是哪一个概念的心智。
2. **提炼旧认知**——听众现在把它装在哪个旧抽屉里。
3. **完成心智隔离**——写出 3–5 条「它不是什么」。
4. **设计小切口**——按优先级取材：亲身故事 > 学员案例 > 创业者困境 > 热点 > 熟悉细节 > 高情绪画面。
5. **打出大纵深**——从五维中选 2–4 维，打穿到结构。
6. **完成新植入**——凝成一句正典定义、一个公式或一个模型。
7. **收束高弧光**——回到成长与初心，完成价值复位。
8. **输出可复用成稿**——落成具体载体：短视频、播客、课程开场、公众号、小红书、销售页、演讲、工作坊、书稿。

## 6. 默认输出格式：七段

一篇标准的原力叙事成稿，默认包含七段：

1. **旧认知隔离**
2. **小切口开场**
3. **大纵深展开**
4. **新认知植入**
5. **高弧光收束**
6. **完整版本**（把前五段串成可直接使用的整稿）
7. **可复用金句**（可独立传播、可回写为下次生产资产的短句）

本知识库的概念叙事篇即为此格式的库内变体：隔离、切口、纵深、植入、弧光五段照打，「完整版本」由文档本身承担，末段落为可复用金句与源置信度。

## 7. 三部曲龙骨挂靠

原力叙事不是自由写作，每一次新植入都必须挂靠在三部曲正典龙骨上，防止叙事漂移：

- **母公理**：[ʸx——做会自我繁殖的事](../00-canon/01-母公理ʸx.md)。
- **总飞轮**：母体 → 原力 → 市场 → 生意 → OS → 母体升级。
- **三部主脊**：资产 = 找到不可复制的母体；创业 = 把原力放进时代与市场；OS = 让每次行动都回写成下一次能力。详见[北极星与三部主脊](../00-canon/04-北极星与三部主脊.md)。
- **北极星**：把创始人的不可复制性，变成可复利、可传承、可系统化的事业生命体。
- **十二模块**：A1 发现母体 / A2 回到母体 / A3 获得原力 / A4 显化原力；B1 原力借势 / B2 品类独创 / B3 模式升维 / B4 壁垒锁定；C1 一纸文脉 / C2 一个大脑 / C3 一张地图 / C4 一条链路。

挂靠的检验法：任何一篇叙事的新植入，都应能回答「它是十二模块中哪一格的投影」「它与母公理是什么派生关系」。答不上来的植入，无论多精彩，都不入正典。
