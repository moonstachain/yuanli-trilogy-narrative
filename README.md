## 本库的写法：每篇都是一次心智迁移

每篇概念文档都不是词条，而是一篇完整的**原力叙事成稿**，固定七段：

```
1. 旧认知隔离   —— 它不是什么（腾空旧心智）
2. 小切口       —— 一个具体、热、近、真的入口（打开人）
3. 大纵深       —— 打穿到时代/历史/理论/商业结构（说服人）
4. 新植入       —— 正典定义 + 核心公式 + 完整原子模块表 + 判断标准（改变人）
5. 高弧光       —— 回到成长、初心与原力觉醒（照亮人）
6. 可复用金句   —— 可直接引用转述
7. 源与置信度   —— 每一条内容有据可查
```
